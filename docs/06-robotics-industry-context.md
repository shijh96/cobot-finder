## 二、关键趋势

### 硬件商品化
- 宇树 R1 降到 $4,900 起，个人可买
- Cobot 价格从 $50K 降到 $25K 区间
- 硬件利润率下降，价值往软件层转移

### 软件层缺失
- 68% 中小企业缺乏机器人工程人才
- 通用 MLOps 工具不适合机器人
- 没有跨品牌的标准化平台
- NVIDIA Isaac 是底层，不是终端用户产品

### VC 投资方向
- 2025 年机器人 VC 投资 $103 亿，比 2024 年多 36%
- 重点方向：仓储自动化、人形机器人、手术机器人
- 软件层投资相对不足

## 四、迪士尼 Olaf 机器人案例（引发本次调研的起因）

### 项目概况
- 迪士尼 Olaf（雪宝）是目前最先进的自主行走角色机器人
- 2026/3/16 NVIDIA GTC 大会首秀，Jensen Huang 主题演讲
- 2026/3/29 巴黎迪士尼乐园 World of Frozen 部署

### 核心技术
- 物理仿真：Kamino Simulator (Disney 专有) + Newton 物理引擎 (开源)
- 强化学习：PPO 算法，50 Hz 策略输出，600 Hz 执行器控制
- 训练数据：《冰雪奇缘》原班动画师的动作参考
- 硬件：单块 RTX 4090 即可训练（4,096 并行环境，85K-95K FPS）
- 创新：热管理自调节、13.5 dB 噪音降低、船上行走平衡

### 开源软件栈
- Newton Physics Engine: 开源 Apache 2.0 (Disney + DeepMind + NVIDIA)
- NVIDIA Warp: 开源 GPU 编程框架
- NVIDIA Isaac Lab: 开源机器人训练框架
- PPO 算法库: RL-Games / RSL-RL (开源)

### 与 Cobot Finder 的关联
- Olaf 案例展示了机器人软件层的巨大价值
- 训练工具民主化了（开源 + 消费级 GPU）
- 但上层应用工具严重缺失
- Cobot Finder 正是填补"帮用户选择和使用机器人"这个空白
